Travel Packages Website Frontend
This project provides a user interface for a travel packages website. It includes sections for displaying travel packages, a photo gallery, and a call-to-action. The frontend is built using HTML,CSS and javascript, with a focus on modern design and user experience.

Features
Travel Packages Section: Displays various travel packages with details such as price, duration, and location.
Photo Gallery: Features a gallery of images from travelers.
Call To Action: Encourages users to contact the company for more information.
Footer: Contains contact information and a subscription form for newsletters.
Project Structure
The project is structured as follows:

index.html: The main HTML file containing the structure of the webpage.
styles.css: The CSS file for styling the webpage (not provided, but implied).
assets/: Directory for images and other assets used in the project.
images/: Contains images for the website, including travel packages and gallery images.
assets/js/script.js: JavaScript file for any custom interactivity (not provided, but implied).
assets/js/: Contains any additional JavaScript files if required.
HTML Code Overview
Main Sections
Travel Packages

HTML Elements:
<section> for the travel packages section.
<ul> for the list of packages.
<li> for each package, including a title, description, metadata (e.g., duration, location), price, and a "Book Now" button.
Photo Gallery

HTML Elements:
<section> for the photo gallery.
<ul> for the gallery list.
<li> for each gallery item, containing an <img> tag for the image.
Call To Action

HTML Elements:
<section> for the call-to-action.
Contains a heading, text, and a "Contact Us" button.
Footer

HTML Elements:
<footer> containing brand information, contact details, a subscription form, and additional links (Privacy Policy, Terms & Conditions, FAQ).
Go To Top Button

HTML Element:
<a> link styled as a button to scroll back to the top of the page.
Usage
Running the Project Locally

Open the index.html file in a web browser to view the website.
Ensure all assets (images, stylesheets, JavaScript files) are correctly placed in their respective directories.
Customization

Modify the HTML structure or CSS styles to fit your needs.
Update the image paths in the <img> tags if your directory structure changes.
Adding More Packages or Gallery Images

To add more travel packages, copy an existing <li> element in the travel packages section and update the details.
To add more images to the gallery, include additional <li> elements with <img> tags in the gallery list.
Dependencies
Ionicons: Used for icons in the website. Included via a <script> tag.
